# tws-app
Interactive Brokers and Trader Workstation API access


# Run Code

1. Download and install Trader Workstation (TWS) from Interactive Brokers.
2. Ensure that the TWS API is enabled in the TWS settings.
3. Virtual Environment Setup:
   ```bash
   python -m venv venv
   source venv/bin/activate  
   ```
4. Install Required Packages:
   ```bash
   pip install -r requierements.txt

5. Run the Application:
   ```bash
   python main.py
    ```
